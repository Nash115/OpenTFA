# OpenTFA
An open-source re-implementation of the famous multiplayer archery game, developed in Rust with Bevy Engine.

OpenTFA is an educational and preservation project aiming to recreate the game feel and local multiplayer mechanics of TowerFall Ascension. This project is coded from scratch using Rust and the ECS Bevy engine.
